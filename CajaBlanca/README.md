Autoestopista Galáctica
🔎 Objetivos 🔎 Crear un juego de temática espacial.

🧩 Requisitos 🧩 El juego se desarrolla sobre un tablero compuesto por 42 casillas y utiliza 3 dados especiales de nueve caras cada uno.

Cada galaxia se identifica mediante una combinación numérica de tres cifras. El jugador inicia la partida en la casilla número 1, situada en la galaxia con código 000.

En cada turno, el jugador lanza simultáneamente los tres dados para generar una nueva combinación de tres números, que determinará la galaxia a la que puede desplazarse.

El avance será posible siempre que la galaxia obtenida esté próxima a la actual, considerándose cercana cuando la distancia entre ambas sea igual o inferior a 4 años luz.

⚠️Riesgos⚠️

Si el jugador llega a la casilla 31, será interceptado por extraterrestres hostiles y retrocederá hasta la casilla 13.

Si cae en la casilla 33, quedará atrapado en un agujero negro y la partida finalizará inmediatamente.

🏁 Final de la partida 🏁 El objetivo es alcanzar la casilla 42. Si el jugador la supera, deberá regresar a la casilla inicial (casilla 1). La partida también termina automáticamente si el jugador cae en el agujero negro.

💻 Tecnología 💻

Java

IDE: Apache NetBeans

Draw.io

🚀 Instrucciones de ejecución 🚀

Abre el archivo del proyecto en Apache NetBeans.

Haga clic en el botón verde de ejecución situado en la parte superior.

Se abrirá la consola y podrás comenzar a jugar.
